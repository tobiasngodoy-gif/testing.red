# Sistema de diagnóstico de red basado en planilla de datos
Este sistema realiza pruebas de conectividad tomando como referencia una planilla que contiene direcciones IP y puertos definidos. En función de esos parámetros, evalúa el estado de cada dispositivo y determina si la conexión es exitosa o presenta fallas.

Además, integra un mecanismo de notificación automática a través de un chatbot de Telegram, el cual envía alertas en tiempo real informando cualquier cambio en el estado de la conexión.
